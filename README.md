🔥 Calories Burnt Predictor



A Machine Learning web application that predicts the number of calories burned during exercise based on user inputs like age, gender, weight, workout duration, heart rate, and body temperature.

📌 Features



\- 🎯 Predict calories burned using ML model

\- 🧠 Trained using XGBoost Regressor

\- 🎨 Beautiful UI built with Streamlit

\- 📊 Real-time prediction

\- 🌐 Deployable as a web app



\---



🧠 Tech Stack



\- Python

\- Streamlit

\- NumPy

\- Scikit-learn

\- XGBoost



\---



📂 Project Structure



app.py              # Main Streamlit app

model.pkl          # Trained ML model

requirements.txt   # Dependencies



\---



⚙️ Installation \& Run Locally



git clone https://github.com/your-username/calories-predictor.git

cd calories-predictor

pip install -r requirements.txt

streamlit run app.py
